    용어 사전 Glossary

    1. 기본이 되는 단어
    종속변수 dependent variable
    내가 측정하려고 하는 결과값입니다. "무엇이 달라지는지"를 보는 변수. 예: 통증 점수, 혈압, 걷는 속도.
    독립변수 independent variable
    결과에 영향을 준다고 의심되는 변수. 연구자가 조작하거나 관찰합니다. 예: 치료 종류, 나이, 운동 시간.
    집단 / 조건 group / condition
    비교하려는 단위. 예: "치료군"과 "대조군"은 서로 다른 집단, "치료 전"과 "치료 후"는 서로 다른 조건.
    표본 · 모집단 sample · population
    모집단은 연구 결과를 일반화하고 싶은 전체 대상, 표본은 실제 참여한 사람들.
    2. 데이터의 종류 (측정 수준)
    연속형 continuous
    숫자가 연속적으로 이어지고 값 사이 간격이 일정한 변수. 평균이 의미를 가집니다. 예: 키, 체중, 혈압, 시험 점수.
    서열형 ordinal
    순서는 있지만 간격이 일정하지 않은 변수. 예: 통증 0–10, "경증/중등도/중증", MMT 등급 0–5.
    명목형 nominal / categorical
    순서 없는 범주로만 구분되는 변수. 예: 성별(남/여), 혈액형, 회복 여부(예/아니오).
    3. 표본의 구조
    독립 표본 independent samples
    비교 집단의 구성원이 서로 다른 사람. 예: 치료군 30명과 대조군 30명.
    대응 표본 paired / repeated
    같은 사람이 두 번 이상 측정됨. 예: 같은 환자의 치료 전·후.
    무작위 배정 random assignment
    참여자가 어느 집단에 들어갈지를 동전 던지기처럼 우연에 맡기는 것. 실험 설계의 기본.
    무작위 표집 random sampling
    모집단에서 표본을 뽑을 때 모두가 똑같은 확률로 뽑히도록 하는 것. 배정과 다릅니다.
    4. 통계 가정과 분포
    정규분포 normal distribution
    좌우 대칭의 종(bell) 모양 분포. 많은 통계가 이것을 가정합니다.
    모수 검정 parametric
    정규분포 등 특정 분포 가정을 전제로 하는 통계. 예: t-검정, ANOVA.
    비모수 검정 nonparametric
    분포 가정을 덜 요구하는 통계. 예: Mann–Whitney, Wilcoxon, Kruskal–Wallis. 작은 표본이나 치우친 분포에 안전합니다.
    등분산 homogeneity of variance
    비교 집단들의 퍼진 정도(분산)가 비슷한 것. Levene 검정으로 확인.
    구형성 sphericity
    반복측정 분산분석에서 요구되는 가정. 위배 시 Greenhouse–Geisser로 보정.
    5. 결과의 성격
    p-값 p-value
    "귀무가설이 맞다고 가정했을 때 이 정도 결과가 나올 확률". 보통 0.05 미만이면 통계적으로 유의하다고 봅니다.
    효과 크기 effect size
    차이·관계가 얼마나 큰지의 지표. Cohen's d, η², r 등. p-값과 함께 보고.
    사후 검정 post-hoc
    ANOVA에서 "차이가 있다"가 나왔을 때 어느 쌍이 다른지 찾는 추가 검정. Tukey, Bonferroni 등.
